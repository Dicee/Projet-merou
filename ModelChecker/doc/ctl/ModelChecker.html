Related Doc: package ctl

class ModelChecker[M <: MetaVariable, N, V <: Value] extends AnyRef

The ModelChecker implements the model-checking algorithm and is able to apply CTL expressions on a graph. To construct the ModelChecker, you need to pass it a GraphNode from which all the nodes can be accessed using only the successors to traverse the graph, and a conversion function returning for a given node (or more precisely, the value held by this node) the set of values this node is likely to introduce in the environment while evaluating a Predicate on it.

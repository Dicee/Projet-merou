    CFGDef, just like CFGDecl represents a C++ declaration. However, while CFGDecl corresponds to an actual declaration, CFGDef only represents the semantic of a declaration and not the declaration itself. Indeed, two CFGDef are considered equal if they declare a variable of the same name and the same type.
